# Customer Review Sentiment & Rating Analysis

## Project Overview

This project focuses on analyzing customer reviews and ratings to understand customer sentiment and customer satisfaction patterns. The analysis helps identify positive and negative feedback trends and generates meaningful business insights from review data.

The dataset contains customer reviews, ratings, review dates, and product-related information collected from an e-commerce platform.

---

## Objectives

* Perform data cleaning and preprocessing
* Analyze customer ratings and review patterns
* Classify customer sentiment into positive, negative, and neutral categories
* Visualize customer feedback trends using charts and graphs
* Generate business insights from customer reviews

---

## Technologies Used

* Python
* Pandas

* Matplotlib
* Seaborn
* Google Colab
* Excel 

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Sentiment Analysis
5. Data Visualization
6. Insight Generation

---

## Key Insights

* Identified the distribution of customer ratings
* Analyzed positive and negative review trends
* Detected common issues in low-rated reviews
* Found customer satisfaction patterns over time
* Created visual dashboards for better business understanding

---

* Sentiment Distribution

## Business Impact

This project helps businesses understand customer opinions, improve customer experience, and make data-driven decisions based on review analysis.
